Timetable

First, you should choose start and end station for desired scheduled route and click SEARCH.

After that table with scheduled routes will be displayed for desired route. Table consists of depature time, arrival time, days of the week the train goes on selected route in that time and button for editting scheduled route. If you click on the Edit button, new window with more details about the scheduled route is opened. Above of the table are Add and remove buttons. By clicking on the Add button, new Window for adding new scheduled route is opened. By clicking on the Remove button, selecte row in table is deleted.
