Wagons

First, you have following options: to add new wagon, edit selected wagon, or delete selected wagon

Edit existing wagon

To edit wagon, firstly you have to select one wagon from the list. Then on the right page you will have form for changing existing wagon attributes.

Then you are able to change the name of the wagon. Also you can change number of seat columns and seat rows inside chosen wagon.

To confirm changes click "Save changes" button.
If everything went right confirmation message will popup

Add new wagon

To add new wagon, click on "Add" button. Then on the right page you will have form for input wagon attributes.

Then you are able to input the name of the wagon. Also you can set number of seat columns and seat rows inside chosen wagon.
(Same as in edit section)

To confirm changes click "Save changes" button.
If everything went right confirmation message will popupup
(Same as in edit section)

Delete wagon

To delete wagon, click on "Delete" button in wanted row.

Then confirmation popup will show up to make sure you want to delete selected wagon.
If you change your mind, select option "No" and changes will be canceled.

Close page

To exit wagon crud page and go back to train crud page, click on "Close" button.
