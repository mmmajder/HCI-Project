Train Routes Map

First, you have following options: show selected route on the map, add new train routes,
edit existing train routes and delete selected train route.

Show train route on map

First, you have to choose starting (From) and ending (To) station
by clicking on combobx and selecting wanted station

Then, in combobox labeled "Routes: " will show up all existing routes from start to end location.
You have to choose specific route you would like to see shown on the map.

After filling out all input fields, you can click "Show route" button.
And map on the right will show up all passing stations with drawn selected route.

Add/Edit route

To add new station, click on "Add" button.
To edit existing station, click on "Edit" button. You must previously choose route.(Steps described in previous section)
On the right will show up form with following instructions.

Delete route

To delete route, click on "Delete" button. You must previously choose route.(Steps described in previous section)

Then confirmation popup will show up to make sure you want to delete selected route.
If you change your mind, select option "No" and changes will be canceled.

Edit Station

Additionally, you have an option to edit stations by clicking "Edit stations" button. This will take you to another page with detailed instructions.
