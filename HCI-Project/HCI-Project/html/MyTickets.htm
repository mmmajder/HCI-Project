My Tickets

This window shows all your bought tickets that are valid.

Which means, all tickets which departure is in the future.

In the Top part of the Window you see two buttons.

Both to work you need first to select a Ticket from the Ticket table in the bottom half of the window.

In case you press a button without selecting the row you will see an error.

Cancel Ticket when pressed cancels the ticket if departure is not sooner than 1 day.

Preview Ticket opens detailed preview of the selected ticket. Which is shown below.

Pressing the Close button takes you back to My tickets window, while pressing the Cancel ticket, cancels the ticket.
