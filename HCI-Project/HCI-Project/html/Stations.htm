Stations

First, you have following options: show selected station or all stations on the map, add new station, and delete selected station.

Show station on map

To show location of wanted station you can click on it in the list on the left side.
Then on the right side of the page you have map, where station will be in focus and marked with pin.

You can also choose option to show all existing stations on the map by clicking on toggle button.

Add new station

To add new station, click on "Add" button.

Then you are able to input the name of the station.

After successful input of new station's name, new RED pin will show up on the map

Now you can drag and drop this pin wherever you want new station to be.
You have an option to add multiple new stations before submitting.

You can also cancel every change you made by clicking "Cancel new stations" button.

When you are happy with station's postion, confirm changes click "Save new station" button.
If everything went right confirmation message will popup, and added station will become ORANGE.

Delete station

To delete station, click on "Delete" button in wanted row.

Then confirmation popup will show up to make sure you want to delete selected station.
If you change your mind, select option "No" and changes will be canceled.

Close page

To exit station crud page and go back to route crud page, click on "Close" button.
