Trains

You have following options: add new train type, edit existing train type, delete existing train.

First, You can choose to edit train type by clicking on one option from given train types, and than choose modification you would like to make.
Or you can add completely new train.

Add/Edit Train

You can input wanted name of train, and it must be unique.

To add new wagon on the train, select one wagon from right list(All Wagons) and drag it over and dorp to the left list (Train's wagons).
You can also change the order of already added wagons in Train's wagon list. To remove wagon from the train, select one wagon from left list(Train's wagons) and drag it over and dorp to the right list (All Wagons).
Train must contain at least 1 wagon, for it to be considered route.

Save/Cancel changes

If you changed your mind and no longer want to edit this train, click on "Cancel" button.
If you are happy with your inputs, click "Save changes" button to complete changes.

Edit Wagons

Additionally, you have an option to edit wagons by clicking "Edit wagons" button. This will take you to another page with detailed instructions.
